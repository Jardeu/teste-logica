# Teste Promobit

## Descrição
Esse repositório possui o teste da Promobit para avaliar as habilidades de lógica.

## Como testar?
Para testar seu código, basta apenas rodar o seguinte comando:
```bash
php vendor/bin/phpunit test --colors
```

O resultado deve ser semelhante ao seguinte:
```bash
PHPUnit 8.5.21 by Sebastian Bergmann and contributors.

.                                                                   1 / 1 (100%)

Time: 26 ms, Memory: 4.00 MB

OK (1 test, 1 assertion)
```
